Keep记事应用
============

利用Angular.js创建一个类似与Google Keep的记事本应用.

## Install Requirements

使用npm安装node.js依赖包

```
npm install
```

使用bower安装前端依赖的第三方文件

```
bower install
```

## How To Develop

创建控制器

```
yo angular:controller myController
```

创建指令

```
yo angular:directive myDirective
```

创建过滤器

```
yo angular:filter myFilter
```

创建服务

```
yo angular:service myService
```

## How To Test and Package

运行测试

```
grunt test
```

预览

```
grunt server
```

打包

```
grunt build
```
